# Vault — Watch Collection

A premium watch collection manager for iOS.

## Features
- **Collection** — Browse and manage your timepieces
- **Analytics** — Track value trends and insights
- **Wishlist** — Save watches you're dreaming about
- **Wear History** — Log daily wear
- **Insurance** — Store and manage documents

## Requirements
- iOS 17.0+
- Xcode 16.0+

## Setup
```bash
brew install xcodegen  # if needed
xcodegen generate
open Vault.xcodeproj
```

Built with SwiftUI. No external dependencies.
